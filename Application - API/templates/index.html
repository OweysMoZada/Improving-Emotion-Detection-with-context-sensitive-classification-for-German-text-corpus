Emotion and Sentiment detection

Author: Oweys Momenzada

Date of deployment: 21.09.2021

 

Attribute:

  • {"sentiment"}  - Beinhaltet "sentiment_valence" welches den sentiment score beschreibt und "sentiment_label", welches einen entgültiges label basierend auf den Score berechnet.
    • {"sentiment_valence"} - Float zwischen -1 und 1
    • {"sentiment_label"} - label mit [negative, likely negative, neutral, likely positive, positive]
  • {"emotion"} - beinhaltet fünf Emotionen: fear, anger, sadness, joy, neutral
    • {"fear", "anger", "sadness", "joy", "neutral"} - Float zwischen 0 bis 1
  • {"is_truncated": Bool} - Wurde der Text gekürzt? Maximal Länge eines Textes sind 150 Wörter.

 

Python-Code zum kopieren:

import requests

url = "secret.url.app/api"

def create_emotions_sentiment(text):
    json_text = {'text' : text}
    pred = requests.post(url,json=json_text)

    return pred.json()

Beispiel:

example = create_emotions_sentiment("Nun der Wetterbericht von heute")

print(example)
